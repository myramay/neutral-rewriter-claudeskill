neutral-rewriter
A Claude skill that rewrites any text in plain, neutral, factual language — no spin, no emotion, no loaded words.
What it does
Paste in any text and get back a clean, affectless version. It removes emotional charge, persuasive framing, and loaded wording, keeping only the core facts and logical content. Think Wikipedia editor, not op-ed columnist.
Works on any content type:

News articles and headlines
Political statements and press releases
Marketing and advertising copy
Corporate memos and HR communications
Social media posts
Legal or policy language
Opinion pieces

How to use
Once installed, trigger it with phrases like:

"Neutralize this"
"Remove the spin"
"Rewrite this like Wikipedia"
"Make this more objective"
"Remove the bias from this"

What gets removed

Emotionally loaded nouns ("crisis," "catastrophe," "triumph")
Charged verbs ("slammed," "blasted," "celebrated")
Evaluative adjectives ("reckless," "stunning," "shameful")
Intensifiers ("unprecedented," "shocking," "deeply")
Tribal framing ("radical left," "elites," "real Americans")
Rhetorical questions and appeals to fear or outrage
Scare quotes and emphasis formatting

What gets kept

All stated facts (who, what, where, when, how much)
Numerical data and statistics
Direct quotations, attributed to speaker
Named entities (people, organizations, places)
Explicit causal relationships

Example
Before:

In a devastating blow to working families, the governor callously signed a bill that slashes vital benefits for the most vulnerable residents of our state.

After:

The governor signed a bill that reduces benefit payments for low-income residents.

Installation

Download neutral-rewriter.skill
Open Claude → Settings → Skills → Install from file
